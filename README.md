# PROIS Backend API

PROIS (Procurement Information System) is the frontend of the Procurement Information System, built using HTML, Bootstrap, jQuery, and JavaScript (ES Modules).
This application focuses on the user interface and interaction for the Items, Suppliers, and Purchasing modules, and is connected to the backend via APIs.

---

## Installation & Setup

### 1. Backend

Make sure you have setup prois-backend. You can findout how to setup here ðŸ‘‰ [PROIS Backend](https://github.com/jonimaulanayusuf/prois-backend).

### 2. Clone repository

```bash
git clone https://github.com/jonimaulanayusuf/prois.git
cd prois
```

### 3. Run with Live Server (Visual Studio Code)

- Open the project folder in Visual Studio Code
- Install the Live Server extension
- Right-click on the index.html file
- Select Open with Live Server

By default, the application will run at:

```bash
http://localhost:5500
```
